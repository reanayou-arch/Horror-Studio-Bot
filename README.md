# Horror-Studio Bot V1

Telegram бот-игра в стиле Horror-Studio.

## Возможности
- Автор создаёт истории (только ADMIN_ID)
- Игроки выбирают историю и играют
- Хранение в SQLite
- До 15 персонажей на историю
